# ProyectoFinalDam1

# Español

Trabajo final del primer año de DAM1, en el que aplicamos los conocimientos adquiridos durante el curso.

El proyecto consiste en una aplicación destinada a la gestión de información sobre alumnos y ciclos formativos. Los datos se almacenan de forma persistente mediante una base de datos, permitiendo conservar la información a lo largo del tiempo.

Dentro de los archivos del proyecto se incluyen diferentes carpetas de interés, documentación sobre la instalación y el uso de la aplicación, así como diagramas que ayudan a comprender su estructura y funcionamiento.

También se proporcionan los ejecutables en formato .exe y .jar, además del código fuente completo del proyecto.
